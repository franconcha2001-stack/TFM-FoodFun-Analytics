# Food&Fun Analytics

Trabajo Fin de Máster
Business Analytics e Inteligencia de Negocio
Universidad Católica de Valencia

## Objetivo

Desarrollar una solución de Business Analytics para FOOD&FUN que permita:

- Analizar el comportamiento histórico de ventas.
- Predecir la demanda futura.
- Estimar la ocupación de los talleres.
- Optimizar la planificación anual mediante Gurobi.
- Visualizar resultados mediante un dashboard interactivo.

## Tecnologías utilizadas

- Python
- Pandas
- Scikit-Learn
- XGBoost
- Gurobi
- Streamlit
- Excel
